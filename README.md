# University UAF Dashboard

A modern, responsive React.js dashboard for the University of Agriculture, Faisalabad (UAF), featuring role-based views for Students, Faculty, and Administrators.

## üöÄ Getting Started

Follow these instructions to get the project up and running on your local machine.

### Prerequisites

Make sure you have **Node.js** installed (version 18 or higher recommended).
You can check if you have it installed by running:
```bash
node -v
```

### Installation

1.  Open your terminal/command prompt.
2.  Navigate to the project directory.
3.  Install the necessary dependencies:

```bash
npm install
```

### Running the Application

To start the local development server:

```bash
npm run dev
```

Target URL: `http://localhost:5173` (or the port shown in your terminal).

### Building for Production

To create an optimized build for deployment:

```bash
npm run build
```

The output files will be in the `dist` folder.
